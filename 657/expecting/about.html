Site Information
  • Background

    This website is a mobile companion to a prototype collection of MODS records using Greenstone Digital Library software. The topic of the digital collection is resources for first-time expectant mothers. This site provides links to the mobile-friendly websites and applications. It also provides access to the resources not formatted for mobile devices as well as a link to the full digital resource guide , which is not formatted for mobil devices (yet).

    This project began as the final project for INFO657 at Drexel University's iSchool. Valerie Florez is the editor of the collection and this website. This website uses the iWebkit framework.
